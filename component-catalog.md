# Component Catalog

## Purpose

Track reusable components and intended usage.

## Core Buckets

- components/backgrounds/: Background variants.
- components/buttons/: Shared button primitives and interactions.
- components/footer/: Footer patterns.
- components/header/: Header and navigation patterns.
- components/hero/: Hero section variants.
- components/icons/: Shared icon components.
- components/page_specific/: Components tied to specific pages.

## Suggested Entry Format

For each component, document:

- Name
- Path
- Props
- Typical use case
- Notes or constraints

## Example Entry

Name: BackToTopButton
Path: components/buttons/BackToTopButton.jsx
Props: none
Use case: Long pages requiring quick top navigation
Notes: Ensure visibility and contrast on all backgrounds
