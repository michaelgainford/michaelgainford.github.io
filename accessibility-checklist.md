# Accessibility Checklist

Use this checklist before shipping visual or content changes.

## Structure

- [ ] One clear H1 per page.
- [ ] Headings follow logical order.
- [ ] Landmark elements are used where appropriate.

## Content

- [ ] Images have meaningful alt text.
- [ ] Decorative images use empty alt text.
- [ ] Link text is descriptive out of context.

## Interaction

- [ ] All interactive controls are keyboard accessible.
- [ ] Focus states are visible.
- [ ] No keyboard traps are introduced.

## Visual

- [ ] Text contrast is sufficient.
- [ ] Content remains usable on small screens and zoomed views.
- [ ] Motion is not distracting and respects reduced motion where used.

## Validation

- [ ] Manual keyboard pass completed.
- [ ] Screen reader spot-check completed for key pages.
