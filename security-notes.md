# Security Notes

## Scope

General security hygiene for a content-focused web project.

## Basics

- Keep dependencies updated.
- Review third-party packages before adding.
- Avoid exposing secrets in source files.

## Content and Data Safety

- Validate external links added to content.
- Avoid embedding untrusted HTML.
- Keep data files free from sensitive information.

## Dependency and Build Hygiene

- Run dependency audits periodically.
- Remove unused dependencies.
- Investigate high severity advisories promptly.

## Incident Notes

Use this file to track security-related findings and remediation steps.
