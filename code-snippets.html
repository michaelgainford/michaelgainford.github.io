MySQL database connection

Snippet allowing connection to a back-end MySQL database. I have included the two below identical variables where I would store the offline development details, and in the second variable store the actual connection details for the online database, simply toggling the active line. You could use just the one line, upload it to your online area, and then when developing offline, with your local user, password etc present, simply add that file to your .gitignore or just be sure to not re-upload the file via FTP, for example, with the offline details present. Of course, this will prevent the connection to your database.

Time-based welcome message

Snippet allowing message to be dynamic based on the time of day

                  function welcome_message() {
                    $t = date("H")-1;
                    if ($t < 12) {
                      echo "Good Morning";
                    }
                    else if ($t < 17) {
                      echo "Good Afternoon";
                    }
                    else { echo "Good Evening"; }
                  }
