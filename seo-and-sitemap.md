# SEO and Sitemap

## Goals

- Ensure each route has clear metadata.
- Keep sitemap current with live routes.
- Prevent broken links and indexing surprises.

## Metadata Checklist

- [ ] Title is unique and descriptive.
- [ ] Description matches page content.
- [ ] Canonical URL is correct when needed.
- [ ] Social preview fields are set where relevant.

## Sitemap Workflow

1. Add or update route files under app/.
2. Confirm route should be indexed.
3. Update sitemap source data if used.
4. Regenerate or update public/sitemap.xml.
5. Verify no stale or broken route entries.

## Verification

- Check key routes in browser.
- Confirm sitemap URLs resolve.
- Run link checks if available.
